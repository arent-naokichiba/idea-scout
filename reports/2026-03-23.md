# Idea Scout Daily Report - 2026-03-23

## 注目の開発ネタ候補

### ★★★★ [HKUDS/LightRAG](https://github.com/HKUDS/LightRAG)
- **ソース**: GitHub Trending / AI
- **要約**: 軽量なRAG（Retrieval-Augmented Generation）フレームワーク。EMNLP2025採択論文の実装で、シンプルかつ高速な検索拡張生成を実現。
- **開発ネタコメント**: BIM/CAD関連の仕様書・規格文書（IFC仕様、建築基準法等）をRAGで検索可能にするツールに活用できる。Revitアドインから自然言語で仕様を問い合わせるアシスタント機能の基盤として有望。

### ★★★★ [louis-e/arnis](https://github.com/louis-e/arnis)
- **ソース**: GitHub Trending / BIM/CAD
- **要約**: 現実世界の地理データからMinecraftの3Dワールドを高精度に自動生成するRust製ツール。OSMデータを活用。
- **開発ネタコメント**: 地理データ→3Dモデル変換のアプローチはBIM/GIS連携に直接応用可能。OSMデータからIFCモデルの周辺環境（道路・建物外形）を自動生成するツールや、都市スケールのコンテキストモデル生成に転用できる。

### ★★★★ [DreamPartGen: Part-Level 3D Generation via Collaborative Latent Denoising](http://arxiv.org/abs/2603.19216v1)
- **ソース**: arXiv / AI
- **要約**: テキストから意味的に構造化されたパーツ単位の3Dオブジェクトを生成する手法。パーツの機能的・意味的構造を考慮した生成が可能。
- **開発ネタコメント**: BIM部品の自動生成に直結する技術。「引き戸」「手すり」等のテキスト記述からRevitファミリの3Dジオメトリを自動生成するツールの研究基盤になりうる。パーツ分解の概念はIFCの部品階層と親和性が高い。
