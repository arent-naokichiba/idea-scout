### ★★★ [Claude Code sends 33k tokens before reading the prompt; OpenCode sends 7k](https://systima.ai/blog/claude-code-vs-opencode-token-overhead)

- 開発ネタコメント: RevitアドインやIFCツール開発でAIコーディングエージェントを使う際のコスト試算に役立つ。トークン効率を考慮したAIツール選定や、社内開発フローへの組み込み判断の材料になる。

### ★★★ [Why write code in 2026](https://softwaredoug.com/blog/2026/07/09/write-code)
